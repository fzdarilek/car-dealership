# car-dealership
* Car dealership school project made in Java. 
* Tested in VSCode
## How to run
* Make sure you have [Maven](https://maven.apache.org/download.cgi) installed on your device
``` 
mvn clean install
java -jar target/my-project-1.0-SNAPSHOT-jar-with-dependencies.jar
```
